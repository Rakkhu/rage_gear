# Rage Road Mode - Run Instructions

## 1. Setup
Runs the installation for all folders.
```powershell
npm install
cd client && npm install
cd ../server && npm install
cd ..
```

## 2. Run
Starts Server (Port 3000) and Client (Port 5173).
```powershell
npm run dev
```

## 3. Play
- Browser: http://localhost:5173
- Mobile: http://<YOUR_IP>:5173

## Features
- **Offline**: Bots (Basic movement)
- **Online**: Invite Code Lobby, Server Physics, Procedural Map
- **Controls**: WASD/Touch, Vehicle Combat.
